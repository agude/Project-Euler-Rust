# Project Euler in Rust

Solutions to [Project Euler](https://projecteuler.net/) problems in Rust.

## Usage

```bash
# Run a solution
cargo run --bin 001

# Run tests
cargo test
```

## Structure

- `src/bin/` - Problem solutions (001.rs, 002.rs, ...)
- `src/utils/` - Shared math utilities (primes, fibonacci, factorization, etc.)
